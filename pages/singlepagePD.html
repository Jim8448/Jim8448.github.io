Single Page Website Project Details

This web page was an assignment for Web Design 2. The goal of this project was to construct a single-page web site that had multiple sections. The user was to be able to navigate to those sections by simply clicking the menu item in the site header. The site also had to have a theme of some kind in either a movie or book style format. I decided on a horror themed film I titled Crimson Midnight. CSS animations were also added in to embelish the UI/UX.
